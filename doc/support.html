2. Support

In order to get the latest driver version please refer to http://www.meilhaus.com.

2.1. Support Requests

As far as known any kernel 2.6.x up to version 2.6.26 is working.
If any problem occurs, please feel free to contact our support.
When starting a support request some essential informations are required:

  • The name and the version of your distribution.
  • The version of your running kernel.
  • The output of 'lspci -vv' and/or 'lsusb -vv'.
  • The output of 'lsmod'.
  • Error messages eventually printed to /var/log/messages.
  • Any error message you got on the terminal.
